# Books I Read - Frontend

React 19 + Vite 7 + Tailwind CSS 4 frontend for the Books I Read platform.

See the [root README](../README.md) for full documentation, features, API reference, and setup instructions.

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run deploy` | Build + Firebase deploy |
